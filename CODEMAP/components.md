# Components / 组件

Short bilingual descriptions of key frontend components.

- AdminBar/
  - EN: Admin bar client component integrating Payload AdminBar UI.
  - CN: 集成 Payload AdminBar 的客户端组件。

- BeforeDashboard/
  - EN: Admin dashboard welcome component and seed button.
  - CN: 管理仪表盘欢迎组件与种子（seed）按钮。

- BeforeLogin/
  - EN: Component shown before admin login.
  - CN: 登录前展示的自定义组件。

- Card/
  - EN: Card UI component for posts or list items.
  - CN: 列表项/文章卡片组件。

- CollectionArchive/
  - EN: Displays a grid/list of Card components for archives.
  - CN: 显示卡片网格/档案列表的组件。

- Logo/Logo.tsx
  - EN: Logo component (SVG image wrapper).
  - CN: Logo 组件（SVG 包装器）。

- Media/
  - EN: Media renderer component used across the site.
  - CN: 媒体渲染组件（站点通用）。
