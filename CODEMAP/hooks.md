# Hooks / 钩子

Short bilingual descriptions of important hooks.

- formatSlug.ts
  - EN: Format slug utility hook.
  - CN: Slug 格式化工具。

- populatePublishedAt.ts
  - EN: Hook to set publishedAt when status transitions to published.
  - CN: 状态变为 published 时自动设置 publishedAt 的 Hook。

- revalidateRedirects.ts
  - EN: Hook to revalidate Next cache tags for redirects.
  - CN: 重新验证重定向缓存标签的 Hook（用于触发 revalidate）。
