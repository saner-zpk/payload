# Blocks / 区块

Key layout blocks used by the layout builder and their bilingual descriptions.

- Banner/
  - EN: Banner block config (style + richText content).
  - CN: 横幅区块配置（样式 + 富文本内容）。

- Code/
  - EN: Code block config (language + code field).
  - CN: 代码区块配置（语言选择 + 代码字段）。

- MediaBlock/
  - EN: Media upload block for embedding media in layouts.
  - CN: 媒体区块，用于在布局中嵌入上传的媒体。

- ArchiveBlock/
  - EN: Archive block used to render lists/archives.
  - CN: 归档/列表区块。

- CallToAction/
  - EN: CTA block config.
  - CN: 呼吁操作（CTA）区块。

- Content/
  - EN: Rich content block (richText with lexical editor features).
  - CN: 内容区块（使用 Lexical 编辑器的富文本）。

- Form/
  - EN: Form block config used with form builder plugin.
  - CN: 表单区块（与表单构建插件配合）。

- RelatedPosts/
  - EN: Block to reference related posts.
  - CN: 相关文章引用区块。

- RenderBlocks.tsx
  - EN: Renderer mapping blocks to frontend components.
  - CN: 前端渲染器：将区块映射到 React 组件。
